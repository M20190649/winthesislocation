all_startend_points:

This program calculates:
A. distance covered, 
B. time travelled, 
C. speed, 
D. start hour, 
E. end hour, 
F. start coordinates and end coordinates

for each trajectory file.

1. Change variable Geolife file source path in variable "file_path"
2. Change the destination path in variable "destination". At this path all the files are created.
3. Change the loop inside <main> "for i in range(0, 1):" to select the users. 
	Example: To run for user 11,12,13,14.,
    		Loop should be: "for i in range(11, 15):"

Run the program to see the results.

offline_staypoints:

This program finds staypoints and snap staypoints to the knows staypoints to form states for a particular user and month in a batch process(offline) to understand the process better.

input:
1. User
2. Month
3. File source path
4. File destination path

output:
1. Offline states with hourly weights


Steps:
1. Change user in variable "user"
2. Change the month in variable "month"
3. Change Geolife source file path in variable "file_src"
4. Change destination files path in variable "dest_path"

Run the program!
